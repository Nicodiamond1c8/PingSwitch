# PingSwitch

Use PingSwitch to on/off your linux (debian) server ping replies.

## Requirements

PingSwitch requires **dialog** package to run:

```sh
sudo apt-get install dialog
```

## Usage

```sh
$ chmod +x pingswitch.sh
$ ./pingswitch
```
